# API reference

This section will hold the **auto-generated** API reference for the FarmUI-Admin
source (TypeScript). Nothing here is written by hand — a generator runs in CI
before the Docusaurus build and emits markdown into this folder.

:::note
The generator is not wired yet. For the backend endpoints the Admin UI consumes
today, see the hand-maintained [API usage](/guides/api_usage) guide.
:::

| Stack | Generator | Emits into |
| --- | --- | --- |
| Python (FarmBE / FarmLink / FarmSimulator) | `pydoc-markdown` | `api-reference/` |
| TypeScript (FarmUI-Admin) | `typedoc` + `typedoc-plugin-markdown` | `api-reference/` |
| JavaScript (FarmUI-Dashboard) | `typedoc` (JS mode via `jsconfig.json`) — thin | `api-reference/` |
| OpenAPI (FarmBE) | `redocusaurus` renders `openapi.json` | a dedicated route |

Once a generator is wired (see the `docs.yml` workflow in
`templates/github-actions/`), switch the sidebar's `api-reference` category to
`{type: 'autogenerated', dirName: 'api-reference'}` so generated pages appear
automatically.
